# Django Web Catalog

Учебный проект веб-приложения для управления каталогом продукции (тематика: "Сыры"). Ключевой особенностью разработки является фокус на обеспечении качества кода через полное покрытие функционала модульными тестами (Unit Testing) и подготовка среды для контейнеризации.

## Технологический стек
* **Фреймворк:** Django
* **Тестирование:** Unittest (стандартная библиотека Python)
* **Развертывание:** Docker (подготовлен Dockerfile)
* **База данных:** SQLite (встроенная БД Django)

## Функциональные возможности

Архитектура приложения построена на четком разделении прав доступа и особой логике ценообразования.

### 1. Ролевая модель доступа
Система поддерживает четыре уровня доступа пользователей:
* **Администратор:** Полное управление номенклатурой товаров (CRUD-операции: создание, чтение, обновление, удаление).
* **Эксперт:** Актуализация и модерация характеристик и описаний товаров.
* **Менеджер:** Формирование оптовых партий для закупки, работа с функционалом расчета стоимости.
* **Пользователь:** Просмотр каталога продукции и использование фильтров.

### 2. Бизнес-логика
* **Динамическое ценообразование:** Реализован алгоритм расчета итоговой стоимости партии с учетом оптовых скидок (зависимость цены от количества единиц товара).
* **Фильтрация:** Система поиска и сортировки товаров по ключевым атрибутам.
* **ORM:** Взаимодействие с базой данных реализовано средствами Django ORM.
